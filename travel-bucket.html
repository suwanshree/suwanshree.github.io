Travel Bucket: Collaborative Trip Planner

Travel Bucket is a Wanderlog-style itinerary builder for planning trips with friends. Users can set dates, save places, add notes, and drag events onto each day to create a plan that stays organized and easy to update. The live demo is available at travel-bucket.onrender.com.

Planning Around Real User Tasks

The interesting part of this project is the interface design challenge hidden inside a full-stack build. Travel planning can get cluttered quickly, so the product has to help people move between dates, locations, notes, and shared decisions without losing the thread of the trip. That means interaction patterns matter just as much as the underlying CRUD.

Focus: Scheduling UX, collaborative planning, and dependable data flows.
Stack: React, Express, PostgreSQL, drag-and-drop style itinerary management, and sharing features.
What it shows: Product-minded front-end development grounded in real planning workflows.

Why It Matters

Travel Bucket shows how I think about software beyond implementation details. The goal is not only to store trip data, but to make planning feel clear, social, and flexible. That combination of UX structure and full-stack delivery is a throughline across much of my software work.
